# Orion Gateway Firmware Notice KB-214

KB-214 describes a firmware issue in Orion Gateway version 4.2.1. The issue can cause intermittent frame drops when Orion Gateway routes high-frequency traffic from WeldVision Camera.

If KB-214 is present, plants using WeldVision Camera through Orion Gateway can experience incomplete weld inspection evidence. Atlas Plant and Nova Plant both use WeldVision Camera through Orion Gateway.

EdgeOps Team owns the KB-214 mitigation plan and must coordinate rollout of the hotfix package.
