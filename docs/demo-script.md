# Demo Script

## 60-second version

1. Load the sample corpus.
2. Ask: `Which plants are affected if Orion Gateway firmware issue KB-214 impacts cameras used for weld quality?`
3. Show that the system returns both plants.
4. Open the Retrieval Inspector and point at the graph-supported chunks.
5. Open Graph Explorer on `Orion Gateway`.
6. Run the benchmark and show hybrid > vector-only.

## What to emphasize verbally

- The graph is not decorative; it changes retrieval behavior.
- ETL is automated with structured outputs, not manually curated.
- The benchmark provides a clean baseline comparison.
- The demo UI exposes the retrieval trace, which makes the system explainable.
